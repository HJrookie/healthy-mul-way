## 项目：PureEat / 净食 健康饮食科普网页
- 基于用户给定的技术栈（Vue 3.5 + Vite 8 + Options API）从零搭建，目录 `/Users/sunpeng/project/project-try/healty-guide`。
- 设计：仿 Apple 极简 + 毛玻璃（backdrop-filter blur），护眼浅米色 #F5F5F7 背景、深灰 #1D1D1F 文字；响应式（手机/平板/桌面）。
- 功能：中英文 ZH/EN 切换（store.js 的 ui.lang）、Hero、六大权威指南玻璃卡片网格（WHO/哈佛/地中海/中国/UK NHS/蓝区）、沉浸式文章阅读器。
- 交互：卡片悬停缩放、滚动淡入（v-reveal 指令 IntersectionObserver）、Hero 加载淡入。
- 图标：用内置 Lucide 风格 inline SVG（Icon.vue），零额外依赖（未用 lucide-vue-next，避免安装风险）。
- 数据结构化便于拓展：`src/data/i18n.js`（UI 文案）、`guidance.js`（指南卡片，每条 {zh,en}）、`articles.js`（文章按 block.type 分块，双语 {zh,en}）。新增文章/指南只需追加数据对象。
- 构建通过：`npm run build` 成功（31 模块，JS gzip ~36kB）；`npm run preview -- --host --port 4173` 已启动，http://localhost:4173/ 返回 200。
